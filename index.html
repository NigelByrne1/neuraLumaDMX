AI Driven Translation of Natural Language into DMX Lighting Control

GitHub Repository

neuraLumaDMX explores how natural language can be translated into stage lighting control using AI. The project focuses on bridging the gap between human intent and the technical requirements of DMX512 lighting systems. By using locally hosted large language models, an untrained user can input a command such as “make the lights calm”, which is transformed into structured DMX channel data and transmitted over a USB DMX interface to control physical lighting hardware in real time. A validation and sanitisation layer ensures that only safe and valid DMX data is transmitted.
